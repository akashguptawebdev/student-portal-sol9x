# Student Portal

A **Student Portal** web application to manage student information efficiently. This portal allows students to view their profile, courses, and other personal details, while admins can manage and track all student records.

## Features

### Student Features
- View personal profile details.
- Edit profile information (name, email, contact, etc.).
- Access course information.
- Responsive and user-friendly interface.

### Admin Features
- View all student details.
- Edit or update student records.
- Role-based access control (Admin vs Student).
- Manage and track student courses.

## Technologies Used

**Frontend:**
- React.js  
- Redux Toolkit (State Management)  
- Tailwind CSS & MUI (UI Components)  

**Backend:**
- Node.js & Express.js  
- MongoDB & Mongoose (Database)  
- JWT (Authentication & Authorization)  
- Bcrypt.js (Password Hashing)  

**Others:**
- Axios (API Calls)  
- React Router (Routing)
